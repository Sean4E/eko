## ğŸ“Š Analyzing Your Data

### Export as CSV
1. Go to https://sean4e.github.io/eko/admin.html
2. Click "Export as CSV"
3. Open in Excel, Google Sheets, or any data analysis tool

### Data Fields
Each response includes:
- `myth-story`: User's personal/cultural myth
- `myth-evokes`: What the myth evokes (array)
- `myth-impact`: How myth affects them
- `emotion-feel`: Desired emotional experience
- `immersive-moment`: Description of immersive experience
- `vibe-preference`: Energy level (1-5 scale)
- `body-control`: Desired magical control
- `interaction-style`: Group size preference
- `spotlight-preference`: Attention level (1-5 scale)
- `music-style`: Music preferences (array)
- `custom-sound`: Custom sound preference
- `biometric-sound`: Body sound type
- `sound-relationship`: Sound interaction (1-5 scale)
- `ar-vision`: Visual creation concept
- `visual-style`: Visual aesthetic preference
- `permanence`: Temporary vs permanent
- `topics`: Topics that matter (array)
- `art-purpose`: Purpose of art
- `message`: Installation message
- `ritual-practice`: Personal/cultural rituals
- `ritual-feelings`: How rituals make them feel (array)
- `participantName`: "Anonymous"
- `timestamp`: ISO 8601 timestamp
- `submittedAt`: Firebase server timestamp
